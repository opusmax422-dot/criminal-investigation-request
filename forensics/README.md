# Forensic Evidence Documentation

This directory contains documented network traffic analysis, surveillance pattern evidence, and investigative timeline data collected through legitimate security monitoring tools.

## Directory Structure

- `CANARY-TOKEN-LOGS.md` - Network traffic observations from Canarytokens deployment
- `SURVEILLANCE-GRID-ANALYSIS.md` - Geographic and temporal pattern analysis
- `CORPORATE-CONNECTIONS.md` - Public records documentation of entity relationships
- `TIMELINE.md` - Chronological sequence of events with evidence references
- `NETWORK-INFRASTRUCTURE.md` - Technical analysis of observed network patterns

## Methodology

All data collected through:
- Legitimate security monitoring (Canarytokens - thinkst/canarytokens)
- Public WHOIS/ASN databases
- Google Analytics on owned properties
- Public corporate records and employment data
